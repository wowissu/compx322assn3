Event CRUD API Documentation

This is a simple HTML file to briefly describe the CRUD API for managing events.

Download postman file

  • GET /events/:id: Retrieve a specific event by ID. 
              
// Retrieve a specific event by ID
GET /events/1
Response Body: 
[
    {
        "id": 1,
        "name": "April Graduation",
        "description": "Graduation is a time to acknowledge and celebrate the success of our students in front of the most important people of all - their whanau, family and friends.",
        "startdate": "2024-04-16 08:30",
        "enddate": "2024-04-16 11:30"
    }
]
  • GET /events?name=:name: Retrieve events by filtering name. 
              
// Retrieve events by filtering name
GET /events?name=April
Response Body: 
[
    {
        "id": 1,
        "name": "April Graduation",
        "description": "Graduation is a time to acknowledge and celebrate the success of our students in front of the most important people of all - their whanau, family and friends.",
        "startdate": "2024-04-16 08:30",
        "enddate": "2024-04-16 11:30"
    }
]
  • POST /events: Create a new event. 
              
// Create a new event
POST /events
Request Body:
{
  "name": "Birthday Party",
  "description": "A fun celebration with friends and family."
  "startdate": "2024-05-30",
  "enddate": "2024-05-30",
}
  • PUT /events/:id: Update an existing event by ID. 
              
// Update an existing event
PUT /events/1
Request Body:
{
  "name": "a new event name",
}

Response Body: 
[
  {
    "id": 1,
    "name": "a new event name",
    "description": "Graduation is a time to acknowledge and celebrate the success of our students in front of the most important people of all - their whanau, family and friends.",
    "startdate": "2024-04-16 08:30",
    "enddate": "2024-04-16 11:30"
  }
]
  • DELETE /events/:id: Delete an event by ID. 
              
// Delete an event
DELETE /events/1
Request Body:
{
  "affectedRows": 1
}
  • DELETE /events: Delete all events. 
              
// Delete all events
DELETE /events
Request Body:
{
  "affectedRows": 5
}

Request and Response Formats

The API accepts and returns data in JSON format.
